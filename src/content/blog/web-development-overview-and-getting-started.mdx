## Getting Started

The first step to getting started is to install <Link to='https://nodejs.org/en/' text='Node.js' />, which is a tool used to run <Link to='https://developer.mozilla.org/en-US/docs/Learn/JavaScript' text='JavaScript'/> code outside of a web browser. It's like a special engine that makes your code work on the computer. It's free and can be used on any computer. This will allow you to write and run code that powers web applications and websites. I recommend using <Link to='https://www.freecodecamp.org/news/node-version-manager-nvm-install-guide/' text='NVM' /> for this as it allows you to easily install and switch between different versions of Node.js. (Just trust me on this one.)

Once you have Node.js installed, you can install <Link to='https://code.visualstudio.com/' text='Visual Studio Code' />. Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS that helps you write and edit your code. It has lots of useful features like highlighting important parts of your code, finding errors, and making it easier to write new code. It's easy to use and has a simple interface that shows you all the files and folders you're working on. With Visual Studio Code, you can write better code and create awesome web apps and websites!

Now that you have Node.js and Visual Studio Code installed, you can start coding. 🥳🎉

This is not a deep tutorial on how to code, but I'll give you a quick overview of the basics of web development. If you want to learn more about web development, I recommend checking out <Link to='https://www.w3schools.com/' text='W3Schools' />, <Link to='https://developer.mozilla.org/en-US/docs/Web' text='MDN Web Docs' /> or <Link to='https://www.freecodecamp.org/' text='FreeCodeCamp' />.

## HTML

HTML stands for **H**yper**T**ext **M**arkup **L**anguage. It is the standard markup language for creating web pages and web applications. HTML is the structure of the website (like your skeleton). It is the foundation of the website and is used to define the content, structure, and layout of a website. HTML is the most basic building block of the web and is used to create web pages and web applications. You do this by using HTML elements which are defined by a start tag, some content, and an end tag.

Here is an example of a HTML page:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My first HTML page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <button>Button clicked <span id="clicks">0</span> times.</button>
  </body>
</html>
```

We now have a basic website, but it doesn't look very good. We need to use CSS to make it look better. If you want to learn more about HTML, I recommend checking out <Link to='https://www.w3schools.com/html/' text='W3Schools HTML Tutorial' />, <Link to='https://developer.mozilla.org/en-US/docs/Web/HTML' text='MDN Web Docs - HTML' /> or <Link to='https://www.freecodecamp.org/learn/2022/responsive-web-design' text='FreeCodeCamp - Responsive Web Design' />.

## CSS

CSS stands for **C**ascading **S**tyle **S**heets. It is a style sheet language used for describing the presentation of a document written in HTML. CSS is used to style the website (like your skin, hair and face?) and can control the layout of multiple web pages all at once. It can also be used to define text styles, background colors, and other aspects of web pages that previously could only be defined in a page's HTML. You do this by using CSS selectors to target HTML elements, then appling the rules to what you selected.

Here is an example of a CSS stylesheet:

```css
body {
  background-color: gray;
}

h1 {
  color: white;
  text-align: center;
}

button {
  background-color: black;
  color: white;
}
```

This CSS stylesheet contains three rules. The first rule sets the background color of the body element to gray. The second rule sets the color of the h1 element to white and the text alignment to center. The third rule sets the background color of the button element to black, and the color of the text to white. We can make this look a little better though.

```css
body {
  font-family: Arial, sans-serif;
  background-color: #c7c7c7;
  margin: 0;
  padding: 0;
}

h1 {
  text-align: center;
  margin-top: 50px;
}

button {
  display: block;
  margin: 50px auto;
  padding: 10px 20px;
  background-color: #ff0000;
  color: #fff;
  font-size: 18px;
  border-radius: 5px;
  cursor: pointer;
  transition: all 0.3s ease-in-out;
}

button:hover {
  background-color: #cc0000; /* slightly darker red on hover */
}

#clicks {
  font-weight: bold;
  margin-left: 5px;
}
```

## JavaScript

JavaScript is a popular programming language that is mainly used to make web pages more interactive (like your brain) and engaging for users. It allows developers to create all kinds of interactive features, such as animations, dynamic forms, and games. In addition to its use on the web, JavaScript is also used in server-side development, desktop applications, and mobile app development. It's a versatile and powerful language that has become an essential part of web development, making it a must-know language for developers.

Here is an example of a JavaScript script:

```js
// Select the button element from the HTML and store it in a variable.
const button = document.querySelector("button");

// Create a variable to keep track of the number of clicks.
let clicks = 0;

// Add an event listener to the button. Everything inside the { } will run every time the button is clicked.
button.addEventListener("click", () => {
  // Increment the number of clicks.
  clicks += 1;

  // Update the text of the button to show the number of clicks.
  document.querySelector("#clicks").textContent = clicks;
});
```

This JavaScript script contains three variables. The first variable is a reference to the button element in the HTML. The second variable is used to keep track of the number of times the button has been clicked. The third variable is a reference to the span element in the HTML. This script contains one event listener. The event listener is added to the button element and will run every time the button is clicked. When the button is clicked, the number of clicks is incremented by one and the text of the span element is updated to show the number of clicks. Now we have a basic website with some styling and interactivity.
